# ArcGIS JavaScript API

This is a demo project:

* **Language**: JavaScript
* **Framework**: None (Vanilla JS)
* **Loading from**: Local (@arcgis/core)
* **Loading strategy**: ESM (import)
* **Bundler**: ViteJS

## Instructions

* Install dependencies: `npm install`
* Run development environment: `npm run dev`
* Build: `npm run build`
